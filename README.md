# all-about-rust

# installation step
```
1. open any terminal in your system
2. run this command 
  - $ curl https://sh.rustup.rs -sSf | sh
3. enter 1 for choose default installation
4. when installation is completed then re-open the terminal and enter below command for check the rust version
  - $ rustc --version // rustc is the rust compiler
  - $ rustup --version // for check the version
  - $ rustup update // check the update
  - $ cargo --version // cargo is the package manager like npm
```
